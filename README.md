# Forest Quest

Forest Quest — полностью Kotlin-игра для Android (только ARM64), сочетающая атмосферное 3D-окружение хвойного леса, сбор монет и мобильные элементы управления (виртуальный джойстик и обзор жестами). Проект не содержит бинарных артефактов и может быть собран на любой машине с JDK 17+.

## Возможности

- Красочное главное меню в мультяшном стиле с кнопками «Играть» и «Настройки».
- Настройки графики (низкое/среднее/высокое качество), влияющие на плотность объектов и детализацию.
- 3D-рендер на OpenGL ES 2.0 с динамическим освещением, высокими елями и большой оптимизированной картой.
- Сбор монет в лесу, подсчёт прогресса и плавная анимация вращения монет.
- Управление, адаптированное под сенсорные устройства: виртуальный джойстик для перемещения и свайпы для обзора.

## Сборка локально

1. Установите JDK 17+ и Android SDK (compile/target SDK 34).
2. Выполните команду:
   ```bash
   ./gradlew assembleRelease
   ```
   Скрипт автоматически скачает `gradle-wrapper.jar` при первом запуске.
3. Итоговый APK появится в `app/build/outputs/apk/release/app-release.apk`.

## Сборка и выдача APK через Docker

В репозитории есть `Dockerfile`, который готовит окружение на базе Ubuntu, собирает релизный APK и поднимает HTTP-сервер (порт 8000) с автоматической выдачей файла.

```bash
docker build -t forest-quest .
docker run -p 8000:8000 forest-quest
```

После запуска контейнера переходите на `http://localhost:8000/` и скачивайте свежесобранный APK.

## Структура проекта

- `app/src/main/java/com/example/forestquest` — Kotlin-код активностей, рендерера и UI-компонентов.
- `app/src/main/res` — ресурсы интерфейса и оформление.
- `gradlew`, `gradlew.bat` — обёртки для Gradle без хранения бинарных файлов.
