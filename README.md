# Forest Collect

Небольшая 3D-игра для Android (arm64), в которой игрок попадает в мультяшный хвойный лес, перемещается с помощью виртуального джойстика и собирает золотые монеты. Проект построен на LibGDX и оптимизирован под сенсорное управление на смартфонах.

## Ключевые особенности
- **Главное меню** с ярким мультяшным оформлением и крупными кнопками «Играть» и «Настройки».
- **Экран настроек** позволяет выбирать пресеты качества графики (низкое, среднее, высокое), влияющие на плотность леса и количество объектов.
- **Игровой процесс**: большая оптимизированная карта с елями, вращающимися монетами и счётчиком собранных трофеев. Камера управляется свайпами по правой части экрана, перемещение происходит через джойстик внизу слева.
- **Управление касаниями** адаптировано исключительно под мобильные устройства, поддерживает одновременные жесты (движение + вращение камеры).

## Локальная сборка APK
1. Установите JDK 11+ и Android SDK (потребуются платформы API 34 и build-tools 34.0.0). Убедитесь, что переменная `ANDROID_SDK_ROOT` указывает на директорию SDK, либо создайте `local.properties` с `sdk.dir=...`.
2. В корне репозитория выполните:
   ```bash
   ./gradlew android:assembleDebug
   ```
3. Готовый APK появится в `android/build/outputs/apk/debug/android-debug.apk`.

## Проверка через Docker
Докер-образ автоматически собирает проект и разворачивает HTTP-сервер для скачивания APK.

```bash
docker build -t forest-collect .
docker run --rm -p 8000:8000 forest-collect
```

После запуска контейнера перейдите по адресу [http://localhost:8000/](http://localhost:8000/), чтобы сразу скачать APK. Путь и имя файла можно переопределить переменными окружения `APK_PATH` и `APK_FILENAME` при необходимости.

## Структура проекта
- `core/` — общая игровая логика, экраны и 3D-мир на LibGDX.
- `android/` — Android-обёртка, AndroidManifest и ресурсы.
- `tools/serve_apk.py` — минималистичный HTTP-сервер, который используется в Docker-образе для раздачи собранного APK.
- `Dockerfile` — рецепт сборки образа с автоматической компиляцией и запуском сервера.

## Запуск игры на устройстве
Соберите APK, установите его на ARM64-устройство (`adb install android-debug.apk`) и наслаждайтесь прогулкой по зимнему лесу с поиском монет!
