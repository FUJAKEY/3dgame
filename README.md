# Forest Adventure 3D (Android ARM64)

Мобильная 3D-игра на Android, полностью оптимизированная под arm64-v8a. Проект включает нативный движок на C++/OpenGL ES, красочное главное меню, настройки графики и полноценный геймплей в хвойном лесу: игрок исследует большую карту, управляя виртуальным джойстиком, и собирает золотые монеты у подножий высоких елей.

## Ключевые особенности
- **Главное меню** с мультяшным дизайном, кнопками «Играть» и «Настройки».
- **Настройки графики** (низкое/среднее/высокое) с генерацией леса и монет под выбранный пресет.
- **Нативный рендерер** на C++17 + OpenGL ES 3.0: детализированные деревья, плавная анимация монет, освещение и большая игровая площадка.
- **Сенсорное управление**, адаптированное для смартфонов: виртуальный джойстик для перемещения, свайпы правой половины экрана для обзора.
- **Счётчик монет и FPS** поверх игрового экрана.
- **Полностью автоматизированная сборка** и Docker-образ с HTTP-сервером, который выдаёт готовый APK.

## Требования к окружению
- JDK 17+
- Android SDK (платформа 33, build-tools 33.0.2, CMake 3.22.1, NDK 26.1)
- Python 3 (для сервера из Docker-образа)

## Сборка вручную
1. Установите Android SDK и необходимые компоненты, пропишите `ANDROID_SDK_ROOT` и создайте `local.properties`:
   ```properties
   sdk.dir=/path/to/android-sdk
   ```
2. В корне репозитория выполните:
   ```bash
   ./gradlew assembleRelease
   ```
3. Готовый APK появится по адресу `app/build/outputs/apk/release/app-release-unsigned.apk`.
   При необходимости можно собрать отладочную сборку (подписанную debug-ключом):
   ```bash
   ./gradlew assembleDebug
   ```
   Она создаст `app/build/outputs/apk/debug/app-debug.apk`.

## Запуск на устройстве
1. Перенесите `app-release-unsigned.apk` (или `app-debug.apk`) на телефон или установите через `adb`:
   ```bash
   adb install -r app/build/outputs/apk/release/app-release-unsigned.apk
   ```
2. Управление:
   - Левая половина экрана: виртуальный джойстик для движения.
   - Правая половина: свайпы для вращения камеры.
   - Собирайте монеты под деревьями — прогресс отображается в HUD.

## Docker-образ
Для воспроизводимой сборки и раздачи APK:
```bash
docker build -t forest-game .
docker run --rm -p 8000:8000 forest-game
```
После запуска перейдите на `http://localhost:8000/` — начнётся скачивание `ForestAdventure.apk`.

## Структура проекта
- `app/src/main/java` — Android-логика (главное меню, активити, джойстик, обработка ввода).
- `app/src/main/cpp` — C++ движок, генерация мира, отрисовка и игровая логика.
- `scripts/serve_apk.py` — HTTP-сервер для отдачи APK.
- `Dockerfile` — сборка образа Ubuntu с Android SDK, Gradle и автосборкой APK.
