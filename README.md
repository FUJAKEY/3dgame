# Forest Coins 3D

Forest Coins — это мобильная 3D-игра на Kotlin для Android (arm64), в которой игрок попадает в сказочный еловый лес и собирает золотые монеты. Проект полностью адаптирован под сенсорное управление: виртуальный джойстик отвечает за перемещение, а свайпы по экрану — за управление камерой.

## Основные особенности
- Красочное главное меню в мультяшном стиле, реализованное на Jetpack Compose.
- Настройки графики (низкое / среднее / высокое качество), сохраняемые между сессиями.
- Оптимизированный OpenGL ES 2.0-рендерер со случайной генерацией леса, мягким освещением и анимацией монет.
- Виртуальный джойстик и управление камерой жестами, адаптированные под смартфоны.
- Сбор монет с отображением прогресса в верхнем левом углу экрана.

## Структура проекта
- `app/src/main/java/com/forestcoins/game/MainActivity.kt` — главное меню и настройки.
- `app/src/main/java/com/forestcoins/game/GameActivity.kt` — игровая сцена и обработка ввода.
- `app/src/main/java/com/forestcoins/game/game` — рендерер, меши и элементы управления.
- `app/src/main/res` — ресурсы (темы, строки, макеты, иконки).

## Требования к окружению
- JDK 17+
- Android SDK (compileSdk/targetSdk = 34)
- Интернет-доступ при первом запуске Gradle (скрипт автоматически скачает `gradle-wrapper.jar`).

## Сборка APK
```bash
./gradlew assembleRelease
```
Готовый APK появится в `app/build/outputs/apk/release/`. Каталог с бинарниками игнорируется `.gitignore` и не попадает в репозиторий.


## Запуск из Android Studio
1. Откройте корневой каталог проекта.
2. Дождитесь синхронизации Gradle.
3. Запустите задачу `app:run` или установите APK на устройство arm64.

## Docker-сборка и раздача APK
В репозитории есть `Dockerfile`, который:
1. Собирает release-APK под arm64.
2. Поднимает HTTP-сервер на порту `8000`, отдающий APK по маршруту `/`.

### Пример запуска
```bash
docker build -t forest-coins .
docker run --rm -p 8000:8000 forest-coins
```
После запуска перейдите по адресу `http://localhost:8000/`, чтобы скачать APK.
