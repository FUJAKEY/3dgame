# Forest Coins 3D

Проект представляет собой Android-игру на Java с использованием движка LibGDX. Игрок попадает в атмосферный хвойный лес, исследует территорию при помощи виртуального джойстика и собирает монеты под деревьями. Игра адаптирована под сенсорное управление на устройствах ARM64.

## Основные особенности
- Главное меню в мультяшном стиле с кнопками **Играть** и **Настройки**.
- Раздел настроек позволяет выбрать один из трёх уровней графики (низкий, средний, высокий).
- 3D-окружение: большие ели, просторная карта, динамическое освещение и анимированные монеты.
- Сенсорное управление: виртуальный джойстик для перемещения и просмотр камеры свайпами по экрану.
- Сбор статистики: счётчик собранных монет и таймер прохождения.

## Сборка
Для компиляции проекта необходимы Android SDK (API 34), платформенные инструменты и Gradle 8+. Минимальный уровень API приложения — 26, целевой — 34. Игра собирается только под архитектуру `arm64-v8a`.

### Основные команды
```bash
# Сборка дебажного APK
./gradlew assembleDebug

# Сборка релизного APK
./gradlew assembleRelease
```

### Docker
Для автоматической сборки и публикации APK можно собрать контейнер:

```bash
docker build -t forest-coins .
docker run --rm -p 8000:8000 forest-coins
```

После запуска контейнера APK будет доступен по адресу `http://localhost:8000/` (начнётся скачивание файла `app-debug.apk`).
